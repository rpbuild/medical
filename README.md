# Story Creek Medical Department

This is a static single-page React app built with CDN imports and Babel standalone. It can be hosted directly on GitHub Pages from the repository root.

## Deploy on GitHub Pages

1. Create a repository on GitHub using this project folder.
2. Push the project files to the `main` branch.
3. GitHub Actions will automatically deploy the site using the workflow in `.github/workflows/pages.yml`.
4. Visit `https://<your-username>.github.io/<repository-name>/` to view the app.

## Local preview

Open `index.html` in your browser or use a local static server.

## Notes

- The app is fully static and runs in the browser.
- It stores data in `localStorage`.
- It uses CDN-hosted React, ReactDOM, Babel, and Tailwind CSS.
